# Environment Variables for GenerateByAI

## Supabase Configuration
```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
```

## OpenAI Configuration
```
OPENAI_API_KEY=your_openai_api_key
```

## How to Get Supabase Keys
1. Go to [supabase.com](https://supabase.com) and create a project
2. Go to Project Settings > API
3. Copy the Project URL and paste as `NEXT_PUBLIC_SUPABASE_URL`
4. Copy the `anon` public key and paste as `NEXT_PUBLIC_SUPABASE_ANON_KEY`
5. Copy the `service_role` secret and paste as `SUPABASE_SERVICE_ROLE_KEY`
